Zenith

Zenith is a server software for Getting Over It with Bennett Foddy

Zenith is a multiplayer server software for Getting Over It with Bennett Foddy that aims to provide plugin support and be compatible with many architectures

Zenith is a fork of Skippeh's Oxide.GettingOverItMP with added features
