AntiDuplicatePlayers

by

luckycdev

Prevents players from creating duplicates of themselves (example: spamming control r and tab to create duplicates does not work with this plugin)

v...

AntiDuplicatePlayers is a plugin that checks joining players' IPs and names, and if they both match a currently online players' IP and name, it kicks the duplicate player
