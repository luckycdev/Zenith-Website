Plugin Development Instructions

  1. Install dotnet 5.0.408
  2. You need to have the server built (see Build Instructions for instructions, or just download the build release)
  3. In a folder, run dotnet new classlib -n PluginName (change PluginName to your plugins name)
  4. Open the csproj in Visual Studio 2019
  5. Right click on Dependencies and click Add Project Reference, then at the bottom right click browse, then select ServerShared.dll from the build from step 2*
  6. Code your plugin (look at the source from other plugins for reference)
  7. Click Build at the top and click Build Solution
  8. Simply add your plugins .dll file from your plugin's bin/Debug/net5.0/ into the servers plugins/ folder!

*While developing some plugins, you may need to add more dependencies. For the Broadcast plugin, I had to add Lidgren.Network.dll and UnityEngine.CoreModule.dll along with ServerShared.dll
